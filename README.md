# kabulcardgame
 This is a python shell implementation of the card game kabul. Intructions for gameplay included. 
 You get 4 cards and you want to get rid of as many of them and get the lowest possible number.
 cards Joker, 1-5 are do nothing except for their value, with joker being 0.
 6-Q each have special moves. K is worth 13 but the king of diamonds and the king of hearts is worth -1

 this program does not fully work. It can serve as a milestone in my work but NOT as a full project. This code was written in 2022.
